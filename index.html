Abstract

โครงการนี้นำเสนอระบบวัดเวลาการตอบสนอง (Reaction Time) ที่ใช้งานง่ายและต้นทุนต่ำ ใช้บอร์ด Arduino UNO R4 ร่วมกับจอ TM1637 เพื่อแสดงผลรูปแบบ s.mmm และส่งผลลัพธ์ผ่าน Serial ให้หน้าเว็บที่พัฒนาโดย Bootstrap + Chart.js และ Web Serial API. ผู้ใช้สามารถบันทึกผลเป็น CSV ดูสถิติ Count/Avg/Best/Worst และกราฟแบบเส้น/ฮิสโตแกรมแบบเรียลไทม์ โดยไม่ต้องมีเซิร์ฟเวอร์ เหมาะสำหรับสาธิตเชิงการศึกษาและงานวิจัยพฤติกรรมการตอบสนอง

Overview

  • บอร์ด: Arduino UNO R4 (WiFi/Minima)
  • จอ: TM1637 (CLK=D3, DIO=D6, VCC=5V, GND=GND)
  • ปุ่ม: D2 ↔ GND (INPUT_PULLUP)
  • LED: D4 (แดง), D5 (เขียว) ผ่าน R 220–330Ω ลง GND
  • พฤติกรรม: รอสุ่มเวลา → ไฟเขียว → กดปุ่มเพื่อจับเวลา (นับเฉพาะตอนกด)

How it works

  1. กดปุ่มเริ่ม → ไฟแดงติด, ระบบรอแบบสุ่ม 1–10 วินาที
  2. ครบเวลา → ไฟเขียวติด (READY)
  3. ผู้เล่นกดปุ่ม → หยุดเวลา, แสดงผล s.mmm บนจอ และส่งบรรทัด [RT] Reaction = <ms> ms ทาง Serial
  4. หน้าเว็บอ่าน Serial ผ่าน Web Serial แล้วบันทึก/คำนวณสถิติ + วาดกราฟ
ใช้เว็บเบราว์เซอร์: Chrome/Edge (Desktop) และเปิดผ่าน HTTPS (เช่น GitHub Pages) หรือ localhost

Test Procedure

1) เตรียมฮาร์ดแวร์
  • ต่อจอ TM1637 → VCC=5V, GND=GND, CLK=D3, DIO=D6
  • ปุ่มกด → D2 ↔ GND และตั้ง INPUT_PULLUP ในโค้ด
  • LED แดง D4 → R 220–330Ω → LED → GND
  • LED เขียว D5 → R 220–330Ω → LED → GND
  • เสียบบอร์ด Arduino UNO R4 เข้าคอมพิวเตอร์ด้วยสาย USB
2) อัปโหลดสเก็ตช์
  • เปิด Arduino IDE เลือกบอร์ดและพอร์ตให้ตรงกับ UNO R4
  • ติดตั้งไลบรารี TM1637Display (ผ่าน Library Manager)
  • อัปโหลดสเก็ตช์ที่ส่งบรรทัดผลลัพธ์รูปแบบ [RT] Reaction = <ms> ms ทาง Serial ที่ 9600 bps
  • หลังบูต จอควรแสดง 0 และไฟทดสอบติดดับสั้น ๆ
3) เปิดหน้าเว็บทดสอบ
  • ไปที่เมนู Reflex Test หรือเปิด reflex.html
  • คลิก Connect → เลือกพอร์ตของบอร์ด → กด Connect
  • สถานะเปลี่ยนเป็น Connected และมี Live Log
  • หมายเหตุ: ใช้ Chrome/Edge (Desktop) และเปิดผ่าน HTTPS (เช่น GitHub Pages) หรือ http://localhost
4) วิธีเล่น/ทดสอบ
  1. กดปุ่มที่บอร์ด 1 ครั้งเพื่อ “เริ่มรอ” (ไฟแดงติด)
  2. รอจนไฟเขียว (READY) ติด—ห้ามก่อนเวลา ไม่งั้น False Start
  3. ไฟเขียวติดแล้วให้กดปุ่มทันที ระบบจะแสดงเวลาตอบสนอง (ms)
  4. ผลจะถูกเพิ่มในตาราง/กราฟอัตโนมัติ
  5. กดปุ่มอีกครั้งเพื่อรีเซตกลับสู่สถานะเริ่มต้นและทดสอบรอบถัดไป
5) บันทึก/ดูสถิติ
  • กด Download CSV เพื่อบันทึกผลทั้งหมดเป็นไฟล์ .csv
  • กล่อง KPI แสดง Count / Average / Best / Worst แบบเรียลไทม์
  • กราฟเส้น = ค่าตามลำดับการกด, ฮิสโตแกรม = การกระจายค่า (ปรับ Bin ได้)
Troubleshooting
  • ขึ้น “Failed to open serial port” → ปิด Serial Monitor ของ IDE และลอง Connect ใหม่
  • ไม่เห็นพอร์ต → ตรวจสาย/ไดรเวอร์ และรีพลักบอร์ด
  • ไม่มี Live Log → เช็กว่าโค้ดพิมพ์บรรทัด [RT] Reaction = <ms> ms ตรงรูปแบบ
  • หน้าเว็บยัง Connect ไม่ได้ → ตรวจว่าเปิดผ่าน HTTPS หรือ localhost

Wiring (Quick Ref)

TM1637
  • VCC → 5V
  • GND → GND
  • CLK → D3
  • DIO → D6
ปุ่ม & LED
  • ปุ่ม D2 ↔ GND (ใช้ INPUT_PULLUP)
  • LED แดง D4 → R → LED → GND
  • LED เขียว D5 → R → LED → GND
