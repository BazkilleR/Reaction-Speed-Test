Reaction Speed Project

โปรเจ็กต์วัดเวลาตอบสนอง (Reaction Time) ด้วย Arduino + จอ 4 หลัก TM1637 และหน้าเว็บ (Web Serial) แสดงผลเป็นตาราง/กราฟ พร้อมบันทึก CSV – ฟรี ไม่ต้องมีเซิร์ฟเวอร์

เริ่มทดสอบรีเฟลกซ์ ดูวิธีทำงาน

ภาพรวม

  • บอร์ด: Arduino UNO R4 (WiFi/Minima)
  • จอ: TM1637 (CLK=D3, DIO=D6, VCC=5V, GND=GND)
  • ปุ่ม: D2 ↔ GND (INPUT_PULLUP)
  • LED: D4 (แดง), D5 (เขียว) ผ่าน R 220–330Ω ลง GND
  • พฤติกรรม: รอสุ่มเวลา → ไฟเขียว → กดปุ่มเพื่อจับเวลา (นับเฉพาะตอนกด)

การทำงาน

  1. กดปุ่มเริ่ม → ไฟแดงติด, ระบบรอแบบสุ่ม 1–10 วินาที
  2. ครบเวลา → ไฟเขียวติด (READY)
  3. ผู้เล่นกดปุ่ม → หยุดเวลา, แสดงผล s.mmm บนจอ และส่งบรรทัด [RT] Reaction = <ms> ms ทาง Serial
  4. หน้าเว็บอ่าน Serial ผ่าน Web Serial แล้วบันทึก/คำนวณสถิติ + วาดกราฟ
ใช้เว็บเบราว์เซอร์: Chrome/Edge (Desktop) และเปิดผ่าน HTTPS (เช่น GitHub Pages) หรือ localhost

การต่อสาย (ย่อ)

TM1637
  • VCC → 5V
  • GND → GND
  • CLK → D3
  • DIO → D6
ปุ่ม & LED
  • ปุ่ม D2 ↔ GND (ใช้ INPUT_PULLUP)
  • LED แดง D4 → R → LED → GND
  • LED เขียว D5 → R → LED → GND
